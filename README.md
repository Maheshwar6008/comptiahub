# 📚 CompTIA A+ Learning Hub

> **CompTIA A+ Core 1 (220-1201) & Core 2 (220-1202) — Complete Study Guide**

A beautiful, interactive learning platform for CompTIA A+ certification with mock tests, case studies, and revision sheets.

🌐 **Live Site**: [comptiahub](https://maheshwar6008.github.io/comptiahub/)

---

## ✨ Features

- 📖 **9 Modules** covering all Core 1 & Core 2 domains
- 📝 **Interactive Mock Tests** with two-chance answer system
- 🛠️ **10 Real-World Case Studies** with step-by-step solutions
- 🧠 **Final Revision Master Sheet** with port numbers & exam tips
- 🎓 **Trainer Mode** with talking points and guided questions
- 🔍 **Search** across all lessons
- 📊 **Progress Tracking** with local storage persistence
- 🌙 **Dark Theme** with amber/orange accents

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm

### Local Development
```bash
npm install
npm run dev
```
Open [http://localhost:3000](http://localhost:3000)

### Build for Production
```bash
npm run build
```

## 📝 Mock Test — Two-Chance System

1. Select your answer
2. ✅ **Correct** → Answer locks with explanation
3. ❌ **Wrong** → Get a **second chance** to pick the right answer
4. ❌ **Wrong again** → Correct answer highlighted with full explanation
5. 🏆 View complete score and review all answers at the end
